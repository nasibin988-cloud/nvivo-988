Comprehensive DRI-Based Nutrient Evaluation System Document
Executive Summary
This document provides a complete, evidence-based nutrient evaluation framework for health and nutrition applications, synthesizing data from the National Academies Dietary Reference Intakes (DRI), FDA Daily Values, WHO guidelines, AHA recommendations, and the Dietary Guidelines for Americans 2020-2025. The framework covers 45+ nutrients across macronutrients, vitamins, and minerals with complete demographic breakdowns for personalized nutrition targeting.
Key Implementation Insights:
•	DRI Framework Hierarchy: Use RDA when available (sufficient evidence for 97-98% population adequacy); use AI when RDA cannot be established; UL defines safety ceiling for supplementation
•	FDA Daily Values serve as standardized reference for food labeling (%DV calculations) based on a 2,000-calorie diet for adults/children ≥4 years
•	Classification Logic: Nutrients fall into three categories—Beneficial (encourage adequate intake), Risk (limit intake or monitor for toxicity), and Neutral (maintain within normal range)
•	Special Population Alerts: CKD patients require phosphorus/potassium/sodium restrictions; pregnancy requires folate/iron/iodine/choline monitoring; elderly need enhanced protein/B12/vitamin D
The 2019 National Academies update introduced the Chronic Disease Risk Reduction Intake (CDRR) category for sodium (2,300 mg/day), replacing the traditional UL concept for this nutrient.

Part 1: DRI Framework Explanation
When to use RDA vs AI vs AMDR
Reference Type	Definition	Application	Example
RDA (Recommended Dietary Allowance)	Average daily intake sufficient for 97-98% of healthy individuals	Use as target for individual intake planning	Vitamin C: 90 mg/day for adult males
AI (Adequate Intake)	Estimated adequate intake when insufficient evidence for EAR/RDA	Use as target when RDA unavailable	Vitamin K: 120 mcg/day for adult males
UL (Tolerable Upper Intake Level)	Highest daily intake unlikely to cause adverse health effects	Maximum threshold for supplementation safety	Vitamin D: 100 mcg/day
AMDR (Acceptable Macronutrient Distribution Range)	Range of intake for macronutrients associated with reduced chronic disease risk	Use for macronutrient distribution planning	Carbohydrate: 45-65% of calories
CDRR (Chronic Disease Risk Reduction)	Intake above which reduction is expected to reduce chronic disease risk	Sodium only—reduce intake if exceeding	Sodium: 2,300 mg/day
Clinical Application of UL
The UL matters clinically when:
1.	Supplementation is involved—food sources rarely exceed UL
2.	Fat-soluble vitamins (A, D, E, K)—accumulate in tissues
3.	Minerals with narrow therapeutic windows—iron, zinc, selenium
4.	Pregnancy—vitamin A (retinol) UL of 3,000 mcg RAE is critical due to teratogenicity
Classifying nutrients by %DRI or %UL
Classification Algorithm:
IF intake < 67% of RDA/AI → "Low" or "Inadequate"
IF intake ≥ 67% AND < 100% of RDA/AI → "Below target"
IF intake ≥ 100% AND < 200% of RDA/AI → "Adequate"
IF intake ≥ 200% of RDA/AI AND < UL → "High"
IF intake ≥ UL → "Excess" (Warning)
IF UL = ND (Not Determinable) → No upper warning; flag only if >400% RDA

    "ckd": "Restrict to <2,300 mg/day for all CKD stages",
    "heartFailure": "Typically <2,000 mg/day"

Part 11: Algorithmic Instructions
Computing Personalized Daily Targets
function getPersonalizedTarget(
  nutrientId: string,
  age: number,
  sex: "male" | "female",
  lifeStage: "standard" | "pregnancy" | "lactation",
  conditions: string[]
): number {
  // 1. Get base DRI value for demographic
  let target = getDRIValue(nutrientId, age, sex, lifeStage);
  
  // 2. Apply condition-specific modifications
  if (conditions.includes("ckd") && ["sodium", "potassium", "phosphorus"].includes(nutrientId)) {
    target = applyCKDModification(nutrientId, target);
  }
  if (conditions.includes("hypertension") && nutrientId === "sodium") {
    target = Math.min(target, 1500); // AHA ideal limit
  }
  if (conditions.includes("elderly") && nutrientId === "protein") {
    target = target * 1.25; // 1.0 g/kg vs 0.8 g/kg
  }
  
  return target;
}
Percent of Target Calculation
function calculatePercentOfTarget(intake: number, target: number): number {
  if (target === 0) return 0;
  return Math.round((intake / target) * 100);
}

function calculatePercentOfUL(intake: number, ul: number | null): number | null {
  if (ul === null || ul === 0) return null;
  return Math.round((intake / ul) * 100);
}
Classification Logic
function classifyIntake(
  intake: number,
  target: number,
  ul: number | null,
  nutrientClassification: "beneficial" | "risk" | "neutral"
): IntakeClassification {
  const percentOfTarget = calculatePercentOfTarget(intake, target);
  const percentOfUL = calculatePercentOfUL(intake, ul);
  
  // Check UL first for safety
  if (percentOfUL !== null && percentOfUL >= 100) {
    return "excess"; // WARNING level
  }
  
  // For RISK nutrients (sodium, saturated fat, added sugars)
  if (nutrientClassification === "risk") {
    if (percentOfTarget < 100) return "adequate";
    if (percentOfTarget >= 100 && percentOfTarget < 150) return "high";
    return "excess";
  }
  
  // For BENEFICIAL and NEUTRAL nutrients
  if (percentOfTarget < 67) return "inadequate";
  if (percentOfTarget >= 67 && percentOfTarget < 100) return "below_target";
  if (percentOfTarget >= 100 && percentOfTarget < 200) return "adequate";
  if (percentOfTarget >= 200) {
    if (ul === null) return "high"; // No UL defined
    return percentOfUL! >= 100 ? "excess" : "high";
  }
  
  return "adequate";
}
Fallback Rules
function getFallbackTarget(nutrientId: string): number | null {
  // Use FDA Daily Value as fallback when DRI unavailable for demographic
  const fdaDV = FDA_DAILY_VALUES[nutrientId];
  if (fdaDV) return fdaDV.adults;
  
  // For nutrients without DV, return null (no target)
  return null;
}

function shouldShowWarning(
  nutrientId: string,
  intake: number,
  conditions: string[],
  lifeStage: string
): string[] {
  const warnings: string[] = [];
  
  // Pregnancy-specific warnings
  if (lifeStage === "pregnancy") {
    if (nutrientId === "vitamin_a_preformed" && intake > 3000) {
      warnings.push("CRITICAL: Vitamin A intake exceeds safe limit for pregnancy (teratogenic risk)");
    }
    if (nutrientId === "folate" && intake < 600) {
      warnings.push("Folate below recommended level for neural tube defect prevention");
    }
  }
  
  // CKD-specific warnings
  if (conditions.includes("ckd")) {
    if (nutrientId === "potassium" && intake > 2500) {
      warnings.push("Potassium intake may need restriction based on serum levels");
    }
    if (nutrientId === "phosphorus" && intake > 1000) {
      warnings.push("Phosphorus exceeds CKD recommendation");
    }
  }
  
  return warnings;
}
